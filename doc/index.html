tooltip.js 文档说明

配置参数说明

表格的参数定义在 jQuery.fn.tooltip.DEFAULTS。

# 配置项 标签项 类型 默认值 可选值 说明
title title string 'Tooltip' 任意字符 显示的内容
enableHtml boolean false
false: 否
true: 是
tooltip内容，是否支持html
placement data-tooltip-placement string 'top-center'
'top': 顶部居中
'top-center': 顶部居中
'left': 左侧
'right': 右侧
'bottom': 底部居中
'bottom-center': 底部居中
'top-left': 顶部居左
'top-right': 顶部居右
'bottom-left': 底部居左
'bottom-right': 底部居右
显示的位置
theme string string
'default': 默认
'white': 白色
'primary': primary主色系
'info': info色系
'warn': warn色系
'danger': danger色系
样式风格
opacity data-tooltip-opacity number 1 任意数字 透明度
enableAnimation boolean true
false: 否
true: 是
是否使用动画效果
delay object||number 
{
    show: 1000,
    hide: 1000
}
为object时: 
{
    show: 1000,
    hide: 1000
}

其中show字段表示显示延时，

hide字段表示隐藏延时
为number时: 任意数值，此时显示和隐藏的动画延时值相同，单位为毫秒
动画延时时间
animation object||string undefined
为object时: 
{
    show: 'fadeIn',
    hide: 'fadeOut'
}

其中show字段表示显示动画效果，

hide字段表示隐藏动画效果
为string时: 任意动画效果样式名，此时显示和隐藏的动画效果相同
动画效果的class名
trigger string 'hover'
'hover': 鼠标浮上
'click': 鼠标点击
或者其他自定义事件
触发显示的事件
keepShowWhenOnTip boolean true
false: 否
true: 是
当鼠标在tip内容上时保持显示，鼠标离开则取消保持显示
keepShowWhenClickTip boolean false
false: 否
true: 是
当鼠标点击tip内容时保持显示(即使鼠标移开)，再点击一下将会取消保持显示
cancelKeepShowWhenClickOtherPlace boolean false
false: 否
true: 是
当鼠标点击非tooltip区域时取消一直显示
positionOffset object undefined 
{
    top: '2px',
    left: '2px'
}
其中top表示向下偏移，left表示向右偏移 		tooltip相对偏移位置，e.g:{top: 2px, left: 2px}，其值可为负数
$container jQuery object undefined 任意存在的jQuery的dom对象 放置展示tooltip的容器，默认以body为容器
template string 
<div class="tooltip-wrap" role="tooltip">
    <div class="tooltip-arrow"></div>
    <div class="tooltip-inner"></div>
</div>

事件

# option事件 jQuery事件 参数 说明
1 beforeShow tooltip.beforeShow $tip

tooltip显示之前的事件

$tip: tooltip的jQuery dom对象
2 afterShow tooltip.afterShow $tip

tooltip显示之后的事件

$tip: tooltip的jQuery dom对象
3 beforeHide tooltip.beforeHide $tip

tooltip隐藏之前的事件

$tip: tooltip的jQuery dom对象
4 afterHide tooltip.afterHide $tip

tooltip隐藏之后的事件

$tip: tooltip的jQuery dom对象
5 onToggle tooltip.toggle $tip

tooltip切换显示、隐藏状态的事件

$tip: tooltip的jQuery dom对象

方法

使用方法的语法：$('#element').tooltip('method', parameter);。

# 名称 参数 说明
1 show 无 使tooltip显示
1 hide 无 使tooltip隐藏
1 toggle 无 使tooltip切换显示、隐藏状态
